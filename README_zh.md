学习 Nmap，一个功能强大的开源工具，用于网络探索和安全审计。本学习路径提供专为网络安全初学者设计的全面 Nmap 课程。遵循结构化的路线图，掌握网络扫描、端口发现和漏洞评估。通过实用的非视频教程和在专用网络扫描演练场中的动手练习，您将获得使用 Nmap 绘制网络图和识别安全风险的真实世界经验。
